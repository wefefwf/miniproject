## 프로젝트 소개
Reeflo는 반려동물과 보호자의 삶을 하나의 흐름으로 연결하는 플랫폼입니다.

일상의 기록부터 소중한 추억, 반려동물의 건강 관리까지
반려생활 전반을 한 곳에서 관리할 수 있도록 설계되었습니다.

Reeflo는 반려동물의 삶을 단순한 데이터가 아닌
보호자와 함께 만들어가는 살아있는 기록으로 바라봅니다.

매일의 순간이 자연스럽게 이어지고,
그 모든 시간이 하나의 생태계가 되도록 — Reeflo.
